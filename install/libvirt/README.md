# Criacao do maquina virtual para a instalacao do SNO

Aqui serao descritos todos os passos necessarios para a criacao da maquina virtual onde sera instalado o SNO. Sera criado um storagepool, uma rede e uma maquina virtual. Ha tres scripts que serao usados para criar cada um dos recursos citados anteriormente, revise cada um dos scripts de exemplo e faca as alteracoes necessarias.

## Pre-requisitos

1. Instalacao do virt-manager/libvirt
2. Executar criacao de storage pool, redes e maquina virtuais no libvirt

## Ordem de criacao dos recursos do libvirt

1. Criar o SP
2. Cria a rede
3. Criar a maquina virtual
